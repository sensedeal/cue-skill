# Cue 快速开始指南

> 5分钟上手，开启你的专属调研助理！

---

## 🚀 5分钟快速开始

### 第一步：启动 Cue

Cue 会在你首次使用时自动欢迎你。如果没有收到欢迎消息，只需发送任意消息即可。

### 第二步：开始第一次研究

发送 `/cue` 命令开始深度研究：

```
/cue 分析宁德时代的竞争优势
```

### 第三步：查看任务状态

研究需要 5-30 分钟，你可以随时查看状态：

```
/ct
```

### 第四步：创建监控

监控你关心的标的，自动推送更新：

```
/cm 比亚迪
```

## 💡 核心功能快速体验

### 1. 深度研究

```
/cue <你的问题>
```

**示例：**
- `/cue 现在适合买宁德时代股票吗？`
- `/cue 分析特斯拉2025年的发展前景`
- `/cue 比较小米和华为的竞争优势`

### 2. 任务管理

| 命令 | 功能 |
|------|------|
| `/ct` | 查看当前任务状态 |
| `/ch` | 查看帮助信息 |

### 3. 智能监控

| 命令 | 功能 |
|------|------|
| `/cm <标的>` | 创建新监控 |
| `/cm` | 查看所有监控 |
| `/cn` | 查看最近通知 |

## ⚡ 智能特性（Phase 1-4 新功能）

### 1. 智能理解
- 自动识别问题类型（是什么/为什么/怎么办）
- 自动评估问题复杂度
- 自动提取关键实体

### 2. 动态框架
- 根据问题复杂度自动调整研究深度
- 简单问题快速回答，复杂问题深度分析
- 关键实体定制研究关注点

### 3. 个性化适配
- 学习你的使用偏好
- 自动适配你的研究习惯
- 持续优化推荐视角

### 4. 智能路由
- 8维度意图分析（类型/领域/时间/复杂度/输出/实体/紧急/技术）
- 自动选择最佳研究视角
- 多阶段智能决策

## 📋 常见问题速查

**Q: 研究需要多长时间？**
A: 简单问题 5-10 分钟，复杂问题 15-30 分钟。

**Q: 每天可以用多少次？**
A: Guest 用户：3次深度研究/日，50次对话/日。Owner 用户：无限使用。

**Q: 如何提高研究质量？**
A: 问题越具体，研究质量越高。建议包含具体标的和时间范围。

**Q: 可以同时进行多个研究吗？**
A: 建议一次一个研究，完成后再开始下一个，以保证质量。
