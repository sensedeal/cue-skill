# Cue 常见问题解答（FAQ）

> 用户最常问的问题，这里都有答案

---

## 🚀 入门问题

### Q1: Cue 是什么？
**A:** Cue 是你的专属调研助理，可以帮你进行深度研究、智能监控和专业分析。不只是给答案，还提供完整的证据链和可溯源的研究过程。

### Q2: 如何开始使用？
**A:** 非常简单！
1. 发送任意消息，Cue 会自动欢迎你
2. 发送 `/cue <你的问题>` 开始第一次研究
3. 查看 [快速开始指南](./QUICKSTART.md) 了解更多

### Q3: 需要配置 API Key 吗？
**A:** 当前环境已配置系统级 API Key，Guest 用户无需配置即可使用！Owner 用户可以配置自己的 API Key。

## 🔍 深度研究相关

### Q4: 研究需要多长时间？
**A:** 根据问题复杂度：
- 简单问题：5-10 分钟
- 中等问题：10-20 分钟
- 复杂问题：20-30 分钟

你可以随时用 `/ct` 查看任务状态。

### Q5: 如何提高研究质量？
**A:** 几个技巧：
- ✅ 问题越具体越好
- ✅ 包含时间范围
- ✅ 明确研究角度
- ✅ 一次一个主题

**好问题示例：**
```
/cue 分析宁德时代2025年Q1财报和竞争优势
/cue 比较比亚迪和特斯拉的技术路线
/cue 从基本面分析茅台的投资价值
```

### Q6: 可以同时进行多个研究吗？
**A:** 建议一次一个研究，完成后再开始下一个，这样可以保证研究质量。

### Q7: 研究失败了怎么办？
**A:** 可能的原因和解决方案：
1. **网络问题** - 稍后重试
2. **问题太宽泛** - 把问题具体化
3. **配额用完** - 明天再试或升级权限
4. **系统问题** - 联系管理员

如果问题持续，请记录错误信息并反馈。

### Q8: 研究视角是怎么选择的？
**A:** Cue 使用智能路由系统（Phase 4）：
1. 分析你的问题类型、领域、时间跨度等8个维度
2. 结合你的历史使用偏好（Phase 3）
3. 多阶段智能决策，选择最佳研究视角
4. 你会在研究报告开头看到【智能分析】部分

## 📊 任务和监控相关

### Q9: 如何查看任务状态？
**A:** 发送 `/ct` 即可查看当前任务状态。

### Q10: 如何创建监控？
**A:** 发送 `/cm <标的名称>`，例如：
```
/cm 比亚迪
/cm 特斯拉
/cm 宁德时代
```

### Q11: 监控多久检查一次？
**A:** 每30分钟自动检查一次，发现重要新闻或重大变化时会主动通知你。

### Q12: 可以创建多少个监控？
**A:** Guest 用户最多 5 个监控，Owner 用户无限制。

### Q13: 如何查看通知？
**A:** 发送 `/cn` 查看最近的监控通知，或者等待主动推送。

## 💰 配额和权限相关

### Q14: 每天可以用多少次？
**A:** 

| 用户类型 | 深度研究 | 免费对话 | 监控数量 |
|---------|---------|---------|---------|
| **Guest** | 3次/日 | 50次/日 | 5个 |
| **Owner** | 无限 | 无限 | 无限 |

### Q15: 配额什么时候重置？
**A:** 每日 00:00（北京时间）自动重置。

### Q16: 如何成为 Owner 用户？
**A:** Owner 用户在白名单中，由管理员配置。如需加入白名单，请联系管理员。

### Q17: 配额用完了怎么办？
**A:** 
1. 等待第二天配额重置
2. 联系管理员申请加入白名单
3. 配置自己的 API Key（Owner 用户）

## 🎯 功能相关

### Q18: Phase 1-4 是什么？
**A:** Phase 1-4 是 Cue 的智能增强优化方案：

| 阶段 | 功能 | 说明 |
|------|------|------|
| **Phase 1** | 基础增强 | 问题类型检测、复杂度评估、实体提取 |
| **Phase 2** | 动态框架 | 多复杂度框架、动态选择、实体定制 |
| **Phase 3** | 个性化 | 用户历史分析、个性化框架、偏好学习 |
| **Phase 4** | 智能路由 | 8维度意图分类、多阶段决策、智能提示词 |

查看 [新功能特性](./FEATURES.md) 了解更多。

### Q19: 智能分析是什么？
**A:** 在研究报告开头，你会看到【智能分析】部分，包括：
- 问题类型
- 研究领域
- 时间跨度
- 复杂程度
- 期望输出
- 紧急程度
- 技术水平
- 路由决策

这展示了 Cue 是如何理解你的问题并选择最佳研究方案的。

### Q20: 可以自定义研究视角吗？
**A:** 目前 Cue 会自动选择最佳视角。未来版本会支持手动指定视角。

### Q21: Cue 提供投资建议吗？
**A:** ⚠️ **重要声明**：Cue 只提供客观分析和信息整理，**不提供投资建议**。所有研究内容仅供参考，不构成任何投资决策依据。投资有风险，入市需谨慎。

## 🔧 技术相关

### Q22: 数据安全吗？
**A:** 是的，非常安全：
- API Key 通过系统环境变量或 OpenClaw Secrets 管理
- 不写入用户工作区文件，确保安全隔离
- 用户数据隔离存储
- 审计日志完整记录

### Q23: 支持哪些平台？
**A:** 目前支持：
- 飞书（主要平台）
- Telegram
- Discord
- 更多平台即将支持

### Q24: 可以在手机上使用吗？
**A:** 可以！通过飞书、Telegram 等手机 App 即可使用。

### Q25: 如何反馈问题或建议？

- 在对话中直接告诉我们
- 提交 Issue 到 GitHub
- 联系管理员
