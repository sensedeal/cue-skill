# Cue 功能使用说明

> 完整的功能详解和使用示例

---

## 📋 命令总览

| 命令 | 功能 | 说明 |
|------|------|------|
| `/cue <问题>` | 深度研究 | 启动专业深度研究 |
| `/ct` | 任务状态 | 查看当前研究任务状态 |
| `/cm [标的]` | 监控管理 | 创建或查看监控 |
| `/cn` | 通知列表 | 查看最近监控通知 |
| `/ch` | 帮助 | 查看帮助信息 |

## 🔍 深度研究（/cue）

### 基本用法

```
/cue <你的研究问题>
```

### 使用技巧

#### 1. 问题越具体越好

✅ **推荐：**
```
/cue 分析宁德时代2025年Q1的财报表现和竞争优势
/cue 比较比亚迪和特斯拉在新能源汽车领域的技术路线
/cue 现在适合买入茅台股票吗？从估值和基本面分析
```

❌ **不推荐：**
```
/cue 分析股票
/cue 什么是投资
/cue 帮我看看
```

#### 2. 包含时间范围

```
/cue 分析2025年新能源汽车行业的发展趋势
/cue 回顾茅台过去3年的财务表现
/cue 展望未来6个月的货币政策走向
```

#### 3. 明确研究角度

```
/cue 从技术面分析比亚迪近期走势
/cue 从基本面分析宁德时代的投资价值
/cue 从产业链角度分析苹果公司的供应链
```

### 研究过程

1. **问题分析**（10秒）- 理解你的问题，识别意图
2. **智能路由**（5秒）- 选择最佳研究视角
3. **信息收集**（3-15分钟）- 多源搜索和验证
4. **深度分析**（2-10分钟）- 专业分析和推理
5. **报告生成**（1分钟）- 生成完整研究报告

### 示例研究

#### 示例1：个股分析
```
/cue 分析宁德时代的竞争优势和投资价值
```

**预期输出：**
- 公司概况和业务模式
- 核心竞争力分析
- 财务表现评估
- 估值分析
- 风险提示
- 投资建议

#### 示例2：行业研究
```
/cue 2025年新能源汽车行业发展前景分析
```

**预期输出：**
- 行业现状和趋势
- 竞争格局分析
- 技术路线对比
- 政策环境分析
- 市场空间预测
- 投资机会

#### 示例3：比较研究
```
/cue 比较比亚迪和特斯拉的优劣势
```

**预期输出：**
- 两家公司对比
- 技术路线对比
- 市场份额分析
- 财务状况对比
- 各自优势和劣势
- 未来展望

## 📊 任务状态（/ct）

### 基本用法

```
/ct
```

### 输出说明

| 状态 | 说明 |
|------|------|
| 🟡 **进行中** | 研究正在进行中 |
| 🟢 **已完成** | 研究已完成，可以查看报告 |
| 🔴 **失败** | 研究失败，请重试 |

### 状态示例

```
📊 任务状态
─────────────────
任务 ID: task_123456789
主题: 分析宁德时代的竞争优势
状态: 🟡 进行中
进度: 45% (信息收集中)
开始时间: 2026-03-10 09:30:00
预计完成: 2026-03-10 09:45:00
```

## 📈 监控管理（/cm）

### 创建监控

```
/cm <标的名称>
```

**示例：**
```
/cm 比亚迪
/cm 特斯拉
/cm 宁德时代
```

### 查看监控列表

```
/cm
```

**输出示例：**
```
📋 监控列表（5/5）
─────────────────────
1. 比亚迪 🟢 活跃
   - 最后检查: 2分钟前
   - 最近通知: 1小时前

2. 特斯拉 🟢 活跃
   - 最后检查: 5分钟前
   - 最近通知: 无

3. 宁德时代 🟢 活跃
   - 最后检查: 10分钟前
   - 最近通知: 3小时前
```

### 监控说明

- **监控频率**：每30分钟自动检查一次
- **触发条件**：发现重要新闻或重大变化
- **通知方式**：主动推送消息通知

## ⚡ 智能特性详解

### 1. 问题类型检测

自动识别5种问题类型：

| 类型 | 说明 | 示例 |
|------|------|------|
| 描述型 | 是什么 | "什么是PE比率？" |
| 分析型 | 为什么 | "为什么宁德时代涨了？" |
| 建议型 | 怎么办 | "现在适合买吗？" |
| 比较型 | 对比 | "A和B哪个好？" |
| 预测型 | 未来 | "未来走势如何？" |

### 2. 复杂度评估

自动评估3级复杂度：

| 级别 | 说明 | 研究时间 |
|------|------|---------|
| 简单 | 事实性问题 | 5-10分钟 |
| 中等 | 需要分析推理 | 10-20分钟 |
| 复杂 | 深度研究 | 20-30分钟 |

### 3. 实体提取

自动提取5类关键实体：

| 类型 | 示例 |
|------|------|
| 公司 | 宁德时代、比亚迪、特斯拉 |
| 行业 | 新能源汽车、半导体、医药 |
| 产品 | iPhone、Model 3、比亚迪汉 |
| 概念 | 人工智能、元宇宙、碳中和 |
| 人物 | 马斯克、王传福、雷军 |

### 4. 研究领域识别

自动识别7大研究领域：

| 领域 | 说明 |
|------|------|
| 金融投资 | 股票、基金、投资 |
| 科技创新 | 技术、产品、创新 |
| 消费零售 | 消费、零售、品牌 |

### 5. 智能路由决策

多阶段智能决策流程：

```
1. 问题类型优先 → 选择基础模式
   ↓
2. 研究领域调整 → 优化模式选择
   ↓
3. 用户偏好融合 → 高置信度时优先用户偏好
   ↓
4. 紧急程度响应 → 紧急请求降低复杂度
   ↓
5. 技术水平适配 → 根据用户水平调整框架
```
